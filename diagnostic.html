Outil de Pré-diagnostic Matériel

Afin d'accélérer la prise en charge de votre appareil, nous utilisons un outil de détection sécurisé via notre partenaire DriversCloud.

⚠️ Avertissement de sécurité important

Le site partenaire DriversCloud se finance par la publicité. Vous pourriez voir des boutons "Download" ou "Start" qui ne font pas partie de l'outil. Ignorez-les. Suivez uniquement les étapes décrites ci-dessous.

Étape 1 : Analyser votre appareil

1. Cliquez sur le bouton ci-dessous pour ouvrir la page de détection.
2. Sur la page DriversCloud, faites défiler la page vers le bas.
3. Cliquez sur le bouton bleu "Démarrer la détection".
4. Installez l'agent de détection. Une page web de résultats s'ouvrira dès la fin de l'analyse.
